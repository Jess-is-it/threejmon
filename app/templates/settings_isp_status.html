ISP Port Status reads live RX/TX traffic from the MikroTik interface assigned in System Settings - Routers - ISP Port Tagging. This is observed bandwidth usage, not the physical link negotiation speed. If traffic stays inside the configured 100M window for the configured duration, the ISP is labeled 100M. If observed traffic rises above the 100M ceiling, it is labeled 1G.

Capacity Detection

Example: with a 90 to 105 Mbps window for 10 minutes, an ISP that stays in that range is treated as a 100M circuit. If the same ISP later shows traffic above 105 Mbps, it is treated as 1G because a 100M circuit should not exceed that ceiling.
Default 4h. If the average peak bandwidth stays inside the 100M min/max range, it is also treated as 100M.
